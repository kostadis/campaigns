# Glossary — Canonical Proper Nouns (Phandalin Campaign)

**Purpose:** Lock down canonical spellings for proper nouns the CampaignGenerator pipeline (`distill.py`, `campaign_state.py`, `planning.py`, `party.py`) tends to drift on. Source priority:

1. The four Essentials Kit modules — **DIP**, **SLW**, **SDW**, **DC** (see *Sources* below). When a name appears here, the book spelling wins.
2. The Phandalin campaign's own homebrewed canon (Sotorra, KP, Iymrith, etc.) — flagged in the "Homebrew canon" section.

If the pipeline emits any spelling not in this file, treat it as drift and either fix in the output or add a `[canonical ← variant]` row to the *Known drift map* at the bottom so future runs can grep-correct it.

**Sources:**
- **DIP** — *Essentials Kit: Dragon of Icespire Peak*
- **SLW** — *Essentials Kit: Storm Lord's Wrath*
- **SDW** — *Essentials Kit: Sleeping Dragon's Wake*
- **DC** — *Essentials Kit: Divine Contention*
