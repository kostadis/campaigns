# Malfunctioning Machine

An ancient device brought to the dwarven fortress by [[aletra-satora]]. 

Intended to improve mining efficiency, the device instead malfunctioned, causing massive spatial distortions, "glitching" phenomena, and general chaos throughout the keep. The machine eventually imploded, which coincided with the cessation of the spatial disturbances.
