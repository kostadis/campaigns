# Candlekeep Murders — module proper-noun inventory

Complete inventory of NPCs, locations, items, factions, and concepts
mined from the module JSON:
`docs/background/1073077-Candlekeep_Murders_-_192_Res_-_26th_Dec.json`.

Includes everything from the *Inside the Great Library* companion
section that ships bundled with the 192-page document.

This is the **source-material list**, not the homebrew. For the OOTA
adaptation's adjusted NPC roster (Polly Pocket / Vaelissa T'sarran /
Marin / Brevin / etc.) see `candlekeep_murders_arc.md`.
