# Gauntlgrym Asset Table (Player-Facing)

Use this table when the party negotiates support in Gauntlgrym.

## Quick Rules

- Choose up to 2 assets per operation (1 Entry + 1 Pivot).
- Each asset starts as Ready.
- At high Heat, repeated use risks Strained/Burned status.
- Every asset has an immediate price or in-city liability (not a post-campaign favor).
