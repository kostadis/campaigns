# VTT transcription corrections — campaign name glossary

Misspellings found in Zoom/Otter VTT transcripts and the canonical
forms they should be corrected to. Use this as a reference for
future VTT cleanup passes.

First catalogued during the cleanup of
`summaries/20260510/meeting_saved_closed_caption-toee.vtt`.

The known-canonical set is augmented at scan time by
`notes/proper_nouns_adventure.txt` (passed as
`find_unknowns.py --extra-known`) — that file is derived from the 5e
T1-4 module JSON and covers module-canon places, deities, and creatures
without inflating this glossary with non-misspellings.
