## Rulings & verified attributions (decided by Kostadis, 2026-06-12)

| Issue | Detail | Ruling |
|---|---|---|
| **Greska vs. Grista** | The module contradicts itself: ch2 (line 761) says the Sleeping Giant is "operated by a surly **orc** named **Greska**"; ch5 (line 2438) says the owner is a surly **dwarf** named **Grista** ✓ both verified in source. | **RULED: Grista the dwarf is campaign canon.** Ch2's "Greska the orc" is treated as a module typo — never introduce Greska as an NPC; if "Greska" appears in any text, map it to Grista. |
| **Pip Stonehill collision** | Module's Pip (Toblen's young son, ch2; ten years old in ch6 quest "Pip's Adventure") collides with the campaign sidekick **Pip Thistlewick**. | **RULED: standing rename rule for all pipelines.** Module "Pip" (Stonehill) → **Tuck Stonehill** everywhere, incl. the ch6 quest ("Tuck's Adventure"). Unqualified "Pip" in transcripts defaults to the sidekick Pip Thistlewick. |
| **Elmar vs. Elmina** | Module canon is **Elmina Barthen**, young human woman ✓ (line 660). The contract handout in `notes/open.md` said "Elmar Barthen". | **RESOLVED:** open.md contract fixed to Elmina (2026-06-12). |
| **Daran Edermath is a drow** | Verified ✓ (line 674): "retired **drow** adventurer… well over five hundred years old." This 2023 revision changed him from the original LMoP half-elf. | Don't "correct" drow→half-elf in any pass. |
| **Reidoth is a woman** | Verified ✓ (line 1376): "Reidoth is an elderly human woman" — changed from male in original LMoP. | Use she/her for Reidoth. |
| **Ander is a halfling** | Verified ✓ (line 2302): "teenage **halfling** commoner." One extraction pass called him human — wrong. | Halfling. |
| **Tamara Stonehill is real** | Verified ✓ (line 2394): Toblen and Trilena have a daughter Tamara (ten, ch5) in addition to son Pip/Tuck. | — |
| **Harbin Wester is human** | Verified ✓ (line 767): "a human banker named Harbin Wester." | — |
