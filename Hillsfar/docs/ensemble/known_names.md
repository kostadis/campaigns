# Hillsfar (DDEX34) — campaign-specific known names

Supplements `docs/background/ddex34-its-all-in-the-blood-inventory.md` (the
published-module proper-noun list). This file holds the **homebrew / campaign-
specific** named entities that the module inventory deliberately excludes —
the PCs, the renamed campaign NPCs (the Auditor Vrax'en, High Cloak Veris…),
and campaign-original items/factions. Bold proper nouns are read as known
individuals by `facts_to_state.py --known-names`.

## Player characters

- **Akritas** — ranger/PC (character file spells it "Akrita")
- **Bramgrim** — fighter/PC; full name **Bramgrim Stoutale**
- **Daein** — bard/PC (character file spells it "Daien")
- **Felkur** — PC; full name **Felkur Oldenwood** (character file spells it "Felkur Olwedond")
- **Owlbear** — the party's owlbear companion

- **Veris** — High Cloak Veris / **High Inquisitor Veris**; campaign inquisitor

- **Glor** — NPC (ch 14-15)
- **Thera Grabelthum** — NPC (ch 15-16)
- **Isthana** — head of the Hillsfar Thieves' Guild
- **Teegan** — escaped slave sibling (module spells it "Tegan")
- **Limping Oric** — NPC (ch 10-11)
- **Charles** — Emerald Enclave contact (ch 1)
- **Malkyn Grenefeld** — prior-arc villain referenced in ch 1

- **Brandyr** — fiendborne fire giant (also in module inventory)
- **Vuhm Yestral** — Red Plume Guard Commandant (also in module inventory)
